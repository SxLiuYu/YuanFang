## 🚀 快速开始

### 1. 下载应用

#### Android
- **下载地址**: https://github.com/SxLiuYu/openclaw-clients/releases
- 或通过 GitHub Actions 下载最新 APK

#### 其他平台
- iOS: 需 Xcode 自行编译
- Web: 直接打开 `web/index.html`
- 智能音箱：运行 `python3 smart_speaker/main.py`

### 2. 首次配置

1. 打开应用
2. 输入 DashScope API Key（阿里云通义千问）
3. 配置家庭服务器地址（可选）
4. 开始使用！

### 3. 配置 API Key

**获取 DashScope API Key**:
1. 访问：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云账号
3. 开通 DashScope 服务
4. 创建 API Key

**在应用中配置**:
- 设置 → AI 配置 → 输入 API Key → 保存

## 📱 功能介绍

### 核心功能（15 个模块）

#### 1. 智能家居统一控制 🏠
**功能**: 控制家中所有智能设备
- 支持平台：米家、涂鸦、天猫精灵、HomeKit
- 场景模式：回家/离家/睡眠/工作
- 能耗统计：实时查看设备用电量

**使用示例**:
```
语音："天猫精灵，打开客厅灯"
手动：首页 → 智能家居 → 点击设备
```

#### 2. 家庭账本 💰
**功能**: 记录和管理家庭收支
- 语音记账："花了 50 块钱吃饭"
- 预算管理：设置月度预算，超支预警
- 分类统计：饼图/柱状图展示

**使用示例**:
```
语音："帮我记一下花了 50 块钱吃饭"
手动：账本 → + 记一笔 → 选择分类 → 输入金额
```

#### 3. 家庭任务板 📋
**功能**: 分配和追踪家庭任务
- 任务分配：指定负责人
- 积分系统：完成任务获得积分
- 排行榜：家庭成员积分排名

**使用示例**:
```
创建任务：任务板 → + 创建 → 洗碗 → 分配给孩子 → 10 积分
完成任务：点击任务 → 完成 → 获得积分
```

#### 4. 智能购物清单 🛒
**功能**: 管理家庭购物
- 语音添加："买 5 瓶牛奶"
- 价格对比：京东/淘宝/拼多多/盒马
- 补货提醒：基于消耗频率智能预测

**使用示例**:
```
语音："买牛奶"
查看：购物清单 → 价格对比 → 选择平台购买
```

#### 5. 天猫精灵 AI 技能 🎤
**功能**: 语音控制所有功能
- 唤醒词："天猫精灵，打开家庭助手"
- 支持：设备控制/查询/记账/任务管理
- AI 对话：基于通义千问大模型

**指令示例**:
```
"天猫精灵，打开家庭助手，今天花了多少钱"
"天猫精灵，打开家庭助手，把客厅灯打开"
"天猫精灵，打开家庭助手，花了 50 块钱吃饭"
```

#### 6. 智能提醒服务 ⏰
**功能**: 自动提醒重要事项
- 天气预警：高温/低温/大风/恶劣天气
- 预算预警：支出超 80% 自动提醒
- 任务提醒：逾期/今日/明日到期
- 健康提醒：喝水/伸展/护眼

**推送方式**: 飞书消息/App 通知

#### 7. 自动化规则引擎 ⚙️
**功能**: 创建自动化场景
- 预设规则：晨间/晚安/离家/雨天
- 自定义规则：时间/设备/天气触发
- 智能推荐：AI 学习习惯自动推荐

**使用示例**:
```
晨间例行：7:00 → 开灯 + 开窗 + 播放新闻
晚安例行：23:00 → 关灯 + 关空调 + 布防
```

#### 8. 家庭成员管理 👨‍👩‍👧‍👦
**功能**: 管理多用户权限
- 4 种角色：管理员/成员/儿童/访客
- 权限管理：8 种权限类型
- 邀请码：5 分钟有效邀请码

**角色权限**:
| 角色 | 权限 |
|------|------|
| 管理员 | 所有权限 |
| 普通成员 | 查看控制设备、记账、管理任务 |
| 儿童 | 查看设备、查看完成任务 |
| 访客 | 只读权限 |

#### 9. 数据同步备份 ☁️
**功能**: 数据安全保障
- 云端同步：阿里云 OSS
- 本地备份：自动/手动备份
- 数据导出：JSON 格式
- 数据导入：支持合并/覆盖

**使用示例**:
```
备份：设置 → 数据备份 → 创建备份
恢复：设置 → 数据恢复 → 选择备份
导出：设置 → 导出数据 → 全部数据
```

#### 10. 智能场景推荐 💡
**功能**: AI 学习用户习惯
- 行为学习：记录用户操作
- 模式分析：时间/设备/场景模式
- 智能推荐：自动生成规则

**推荐示例**:
```
检测到每天早上 7 点开灯
→ 推荐：创建 7:00 自动开灯规则（置信度 85%）
```

#### 11. 数据可视化图表 📊
**功能**: 直观展示数据
- 支出饼图：分类占比
- 收支趋势：6 个月趋势线
- 分类柱状图：对比分析
- 积分排行榜：横向条形图

**查看方式**: 账本/任务板 → 统计图表

#### 12. 设备数据服务 📈
**功能**: 设备数据分析
- 状态历史：记录设备变化
- 能耗统计：总计/平均/最大/最小
- 房间统计：设备分布

#### 13. 电商价格爬取 💹
**功能**: 全网比价
- 支持平台：京东/淘宝/拼多多/盒马
- 价格对比：自动找出最低价
- 降价提醒：设置目标价

**使用示例**:

#### 14. 单元测试 ✅

## ⚙️ 配置指南

### 必需配置

#### 1. DashScope API Key

**获取**: https://dashscope.console.aliyun.com/

#### 2. 家庭服务器（可选）
**用途**: 数据同步、多设备共享
**配置**: 设置 → 服务器配置 → 输入地址

### 可选配置

#### 3. 米家 API
**用途**: 控制米家设备
**获取**: https://iot.mi.com/
**配置**: 智能家居 → 添加平台 → 米家

#### 4. 涂鸦 API
**用途**: 控制涂鸦设备
**获取**: https://iot.tuya.com/
**配置**: 智能家居 → 添加平台 → 涂鸦

#### 5. 天猫精灵 API
**用途**: 天猫精灵语音控制
**获取**: https://open.tmall.com/

**用途**: 接收提醒通知

## 📖 使用教程

### 教程 1: 语音记账

1. 打开应用或唤醒天猫精灵
2. 说："花了 50 块钱吃饭"
3. AI 自动识别金额和分类
4. 确认记录

**进阶**:
- "花了 200 块钱买衣服" → 购物分类
- "加油花了 300" → 交通分类
- "今天收入 500" → 收入类型

### 教程 2: 创建自动化规则

**方法 1: 使用预设**
1. 自动化 → 预设规则
2. 选择：晨间例行/晚安例行/离家模式
3. 开启开关

**方法 2: 自定义规则**
1. 自动化 → + 创建规则
2. 设置触发条件：
   - 时间：每天 7:00
   - 设备：客厅灯关闭
   - 天气：下雨
3. 添加执行动作：
   - 控制设备：打开空调
   - 发送通知：提醒带伞
4. 保存

**方法 3: AI 推荐**
1. 自动化 → 智能推荐
2. 查看 AI 学习的习惯
3. 接受推荐 → 自动创建规则

### 教程 3: 管理家庭成员

**添加成员**:
1. 成员管理 → + 添加成员
2. 填写信息：姓名、昵称、头像
3. 选择角色：管理员/成员/儿童/访客
4. 保存

**邀请成员**:
1. 成员管理 → 生成邀请码
2. 分享邀请码给家人
3. 家人输入邀请码加入

**切换成员**:
1. 点击当前成员头像
2. 选择要切换的成员
3. 权限即时生效

### 教程 4: 数据备份与恢复

**创建备份**:
1. 设置 → 数据备份
2. 点击"创建备份"
3. 等待完成（约 10 秒）
4. 备份保存到本地/云端

**恢复备份**:
1. 设置 → 数据恢复
2. 选择备份文件
3. 确认恢复
4. 等待完成（约 20 秒）

**导出数据**:
1. 设置 → 导出数据
2. 选择类型：全部/账本/任务/设备
3. 保存为 JSON 文件

**导入数据**:
1. 设置 → 导入数据
2. 选择 JSON 文件
3. 选择模式：合并/覆盖
4. 确认导入

### 教程 5: 查看统计图表

**支出统计**:
1. 账本 → 统计
2. 选择时间：本月/本年/自定义
3. 查看图表：
   - 饼图：分类占比
   - 趋势图：月度变化
   - 柱状图：分类对比

**任务统计**:
1. 任务板 → 统计
2. 查看：
   - 任务状态分布
   - 成员完成率
   - 积分排行榜

**设备统计**:
1. 智能家居 → 统计
2. 查看：
   - 设备在线情况
   - 能耗趋势
   - 房间分布

## ❓ 常见问题

### Q1: API Key 在哪里获取？
**A**: 访问 https://dashscope.console.aliyun.com/，注册后创建 API Key。

### Q2: 如何绑定天猫精灵？
**A**: 
1. 在天猫精灵技能平台创建技能
2. 配置 Webhook 地址
3. 在应用中配置天猫精灵 API

### Q3: 数据会丢失吗？
**A**: 不会。数据自动备份，支持云端同步。建议定期导出备份。

### Q4: 支持哪些智能设备？
**A**: 支持米家、涂鸦、天猫精灵、HomeKit 等主流平台。

### Q5: 如何多设备共享数据？
**A**: 配置统一的家庭服务器地址，数据自动同步。

### Q6: 儿童模式有什么限制？
**A**: 儿童只能查看设备和任务，不能控制设备或记账。

### Q7: 预算预警如何设置？
**A**: 账本 → 预算 → 设置分类预算 → 开启预警（默认 80% 预警）

### Q8: 智能推荐准确吗？
**A**: 基于 AI 学习，使用 7 天后准确率可达 85% 以上。

### Q9: 如何关闭自动备份？
**A**: 设置 → 数据备份 → 自动备份 → 关闭

### Q10: 支持离线使用吗？
**A**: 支持。基础功能可离线使用，AI 功能需要网络。

## 🔌 API 参考

### 后端 API

**基础 URL**: `http://localhost:8082`

#### 设备管理
```
GET  /api/smarthome/devices     # 获取设备列表
POST /api/smarthome/device      # 添加设备
POST /api/smarthome/control     # 控制设备
POST /api/smarthome/scene       # 执行场景
```

#### 家庭账本
```
POST /api/finance/transaction   # 添加交易
GET  /api/finance/stats         # 获取统计
POST /api/finance/budget        # 设置预算
```

#### 任务管理
```
GET  /api/tasks                 # 获取任务
POST /api/tasks                 # 创建任务
POST /api/tasks/:id/complete    # 完成任务
GET  /api/leaderboard           # 获取排行榜
```

#### 购物清单
```
GET  /api/shopping/list         # 获取清单
POST /api/shopping/item         # 添加物品
POST /api/shopping/item/:id/purchase # 标记购买
GET  /api/shopping/compare      # 价格对比
```

### 请求示例

```bash
# 添加设备
curl -X POST http://localhost:8082/api/smarthome/device \
  -H "Content-Type: application/json" \
  -d '{
    "device_id": "MI_001",
    "device_name": "客厅灯",
    "device_type": "light",
    "platform": "mihome",
    "room": "客厅"
  }'

# 添加交易
curl -X POST http://localhost:8082/api/finance/transaction \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 50,
    "type": "expense",
    "category": "餐饮"
  }'

# 创建任务
curl -X POST http://localhost:8082/api/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "task_name": "洗碗",
    "assigned_to": "孩子",
    "points": 10
  }'
```
