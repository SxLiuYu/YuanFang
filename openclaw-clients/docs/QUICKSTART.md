# OpenClaw Clients 快速开始指南

**5 分钟上手家庭助手！**

---

## 🚀 第一步：下载应用（1 分钟）

### Android 用户
1. 访问：https://github.com/SxLiuYu/openclaw-clients/releases
2. 下载最新 APK
3. 安装应用

### 其他平台
- **Web**: 打开 `web/index.html`
- **智能音箱**: `python3 smart_speaker/main.py`
- **iOS**: Xcode 编译（开发者）

---

## ⚙️ 第二步：配置 API Key（2 分钟）

### 获取 DashScope API Key
1. 访问：https://dashscope.console.aliyun.com/
2. 注册/登录阿里云
3. 开通 DashScope 服务
4. 创建 API Key（格式：`sk-xxxxxxxx`）

### 在应用中配置
1. 打开应用
2. 进入 **设置** → **AI 配置**
3. 输入 API Key
4. 点击 **保存**

✅ 配置完成！现在可以使用 AI 功能了！

---

## 🎤 第三步：开始使用（2 分钟）

### 方式 1: 语音控制
**唤醒天猫精灵**:
```
"天猫精灵，打开家庭助手"
```

**常用指令**:
```
"今天花了多少钱"        # 查询账本
"把客厅灯打开"          # 控制设备
"花了 50 块钱吃饭"       # 语音记账
"有哪些待办任务"        # 查询任务
"买牛奶"               # 添加购物
```

### 方式 2: 手动操作

#### 🏠 智能家居
- 首页 → 智能家居
- 点击设备开关
- 选择场景模式（回家/离家）

#### 💰 家庭账本
- 底部导航 → 账本
- 点击 **+ 记一笔**
- 输入金额和分类

#### 📋 家庭任务
- 底部导航 → 任务板
- 点击 **+ 创建任务**
- 分配给家庭成员

#### 🛒 购物清单
- 底部导航 → 购物清单
- 点击 **+ 添加物品**
- 查看价格对比

## 💡 进阶技巧

### 1. 创建自动化规则
```
自动化 → + 创建 → 7:00 开灯 → 保存
```

### 2. 设置预算预警
```
账本 → 预算 → 餐饮 2000 元 → 开启预警
```

### 3. 添加家庭成员
```
设置 → 成员管理 → + 添加 → 生成邀请码
```

### 4. 数据备份
```
设置 → 数据备份 → 创建备份
```
