## Features

- Device Login Authentication
- Home Dashboard
- Voice Control
- Health Management
- Finance Management
- Task Management
- Configuration Persistence

## Development Environment

- .NET 8.0
- WPF
- MaterialDesignThemes
- CommunityToolkit.Mvvm

## Build & Run

```bash
cd windows-desktop/OpenClaw.Desktop
dotnet build
dotnet run
```
